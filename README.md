# AutomationPracticeTests
Katalon Automation Function
Chosen Automation Function: 
I have chosen Katalon for running the automated tests and wrote my tests in
apache Groovy language and  with Java compatiblesyntax.


This repository contains automation scripts for testing various functionalities on the Automation Practice website using Katalon Studio.

Pre-conditions

Download Katalon Studio

How to Run the tests

1. Clone the repository and open the project in Katalon Studio.
2. Open the test case you want to run.
3. Click on the Run button to execute the test case.
